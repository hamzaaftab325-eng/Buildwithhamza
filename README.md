# Build with Hamza

Modern portfolio website built with Next.js 15, TypeScript, Tailwind CSS, and GSAP.

## Features
- Animated navbar with scroll effects
- Dark hero section with background image
- Responsive design
- GSAP animations
- Deployed on Vercel

## Tech Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- GSAP
- Framer Motion
